# Preview Controls

この文書は、プレビュー周辺の操作だけをまとめた短い説明です。

## Layout

preview 周辺は、desktop / tablet と mobile で少し構成が違います。

desktop / tablet:

- 左: 3D プレビュー
- 右: モード選択と表示トグルのカード
- プレビュー右下: 視点操作まわり
- プレビュー左下: 操作説明

mobile:

- 上: 3D プレビュー
- その下: `基本 / 面 / 双晶 / 表示 / 出力` タブ
- プレビュー下端: 小さい視点操作帯
- 操作説明: canvas の外側

## Camera Operations

プレビューでは次の操作ができます。

desktop / tablet:

- 左ドラッグで回転
- 右ドラッグで平行移動
- ホイールで拡大縮小
- `視点リセット`

mobile:

- 1本指で回転
- 2本指で拡大縮小
- `視点リセット`

`視点リセット` を押すと、現在の視点を初期表示相当へ戻します。

## Inertia

`回転の慣性` をオンにすると、ドラッグ後もしばらく慣性で回転します。

mobile では、1本指回転のあとに指を離した時だけ慣性が続きます。2本指操作を含む touch gesture の直後には、回転中心がずれにくいよう慣性を始めません。

## Axis View Buttons

軸方向ボタンでは、代表的な向きへ素早く視点を合わせられます。

用途:

- 結晶の主軸方向を確認したい時
- 面が正対しているか見たい時
- スクリーンショット用に向きを揃えたい時

## Display Toggles

右カードの表示トグルでは、必要な情報だけを表示できます。

- 面指数
- 稜線
- 交線
- 軸（内）
- 軸（外）
- 軸ラベル
- 双晶軸 / 双晶面ガイド
- 分割面
- ラベル
- 各結晶の表示

`分割面` は既定ではオフです。`STL分割を有効にする（β版）` で設定した平面を preview 上にガイドとして表示したい時だけオンにします。

面文字が入っている面では、文字と面の境界が補助線として重なって表示されます。これは双晶の `交線` トグルとは独立して、面文字の位置や大きさを確認しやすくするための表示です。

mobile では、これらの表示トグルと `表示対象の結晶` は `表示` タブにまとまっています。

また、スマホ幅では面指数以外の preview 文字が自動で少し小さく表示され、軸ラベル、双晶ラベル、preset 情報が重なりにくいようにしています。

## Display Modes

右カードの一番上では表示モードを選べます。

### grouped

- 面ごとの分類が見やすい表示
- 面の違いを見たい時に向きます

### solid

- 結晶色ベースの単色表示
- 全体形状を確認しやすい表示です

### white

- 白ベースの表示
- 稜線や文字とのコントラストを見たい時に向きます

### xray 系

- 半透明や透過を使った表示
- 内部稜線や重なりを確認したい時に向きます

### custom

- `プレビュー詳細設定` で調整した内容を使う表示

## Preview Detail Settings

面一覧カードの下にある `プレビュー詳細設定` では、表示をさらに細かく調整できます。

たとえば次を変更できます。

- ラベルフォント
- ラベル色
- 軸線色
- 稜線色
- 交線色
- 各線の透明度
- custom face/line profile

mobile では `表示` タブの下部に出て、`面指数と双晶 / 軸ラベル / プリセット表示 / 稜線と交線 / 軸線` の順でまとまっています。

文字サイズの既定値自体は desktop 基準を維持し、スマホでは表示時だけ面指数以外を小さめに補正します。

## Tips

- 形の確認には `solid`
- 面の違いを見るには `grouped`
- 内部の重なりを見るには `xray` 系
- 最終的な見せ方を詰めるなら `custom`

の順で使うと分かりやすいです。
