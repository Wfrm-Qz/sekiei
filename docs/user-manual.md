# User Manual

この文書は、Sekiei を初めて使う人向けの操作ガイドです。

公開版は https://sekiei.pages.dev/ から利用できます。

スクリーンショットは基本的な画面構成を把握するためのものです。UI が更新されて見た目や配置が変わった場合は、必要に応じて画像を差し替えます。

## 1. できること

Sekiei では、次のような作業ができます。

- 結晶パラメーターから 3D モデルを作る
- 双晶構成を作る
- 面ごとの色や文字加工を設定する
- ブラウザ上で 3D プレビューする
- STL / SVG / PNG / JPEG / JSON を保存する

![デスクトップ画面の全体。左に入力欄、中央に3Dプレビュー、右に表示設定があります。](./images/user-manual/desktop-overview.png)

## 2. 基本的な流れ

最初は次の順番で使うのがおすすめです。

1. プリセットを選ぶ
2. 結晶パラメーターを調整する
3. 必要なら面一覧で結晶を追加する
4. 双晶パラメーターを調整する
5. 面一覧で面の係数、色、文字加工を調整する
6. プレビューで形状を確認する
7. 出力形式を選んで保存する

スマホでは、プレビューの下に `基本 / 面 / 双晶 / 表示 / 出力` タブが並びます。

- `基本`
  - プリセット、結晶パラメーター、STL分割
- `面`
  - 面一覧と面文字設定
- `双晶`
  - 双晶パラメーター
- `表示`
  - 表示モード、表示トグル、表示対象の結晶、プレビュー詳細設定
- `出力`
  - JSON 保存 / 読み込み、STL / SVG / PNG / JPEG

![スマホ画面の基本タブ。プレビューの下に編集カテゴリのタブが並びます。](./images/user-manual/mobile-basic.png)

## 3. プリセットを使う

画面上部のプリセット欄から、よく使う形状を選べます。

- プリセットを選ぶと、結晶系、軸比、軸間角、面一覧などが入力されます
- プリセット選択後も、表示モードや表示トグルは現在の設定を維持します
- プリセット選択後に、各値をそのまま編集できます

![プリセット欄で quartz と入力し、候補から石英プリセットを選べる状態。](./images/user-manual/preset-search.png)

## 4. お知らせを見る

起動時には、更新履歴と既知の問題をまとめた `お知らせ` が表示されることがあります。

- 一度閉じた内容は、同じ更新日のままでは自動再表示しません
- ヘッダーの `お知らせ` ボタンからいつでも開き直せます
- ヘッダー左上の GitHub / X アイコンから、レポジトリや作者の案内先を開けます
- スマホでは、`お知らせ` と言語切り替えはヘッダー右上の `メニュー` からも開けます

## 5. マニュアルを見る

このマニュアルは、ツール内からも確認できます。

- デスクトップでは、ヘッダーの `マニュアル` ボタンから開けます
- スマホでは、ヘッダー右上の `メニュー` から `マニュアル` を選びます
- マニュアルは画面上にモーダルで表示され、作業中の入力内容はそのまま残ります

## 6. 結晶パラメーター

### 結晶系

結晶系を選ぶと、その結晶系に対応した入力と表示に切り替わります。

### モデルサイズ

- 単位は mm です
- 生成されるモデル全体の最大寸法の目安になります

### STL分割（β版）

結晶パラメーターカードの下部には、STL 保存時だけに使う `STL分割（β版）` があります。

- `分割を有効にする` をオンにすると、通常の STL 保存時に分割処理を行います
- 分割平面は、結晶1の中心を通る指定面指数の平面です
- 指定した面で分割した STL を保存できます
- STL分割の設定は、JSON 保存には含まれません

### 軸比 / 軸間角

- `a / b / c` の軸比を入力します
- `alpha / beta / gamma` の軸間角を入力します
- 結晶系によって一部の値は固定または連動します

## 7. 双晶パラメーター

面一覧で結晶を追加したあと、双晶パラメーターカードで追加した結晶の双晶設定を調整します。

### 主な項目

- 生成元結晶
- 双晶タイプ
  - 接触双晶
  - 貫入双晶
- 双晶面または双晶軸
- 回転角
- 基準方向

### 接触双晶

- 接触面どうしを合わせるタイプです
- 生成元結晶側と派生結晶側の接触面参照を使います

### 貫入双晶

- 双晶軸と回転角で配置するタイプです
- 双晶軸まわりに回転した結晶を重ねて配置します

## 8. 面一覧

面一覧では、各結晶の面を調整できます。

### 結晶タブ

- 左側のタブで対象結晶を切り替えます
- `+` ボタンで結晶を追加します
- タブ右側のメニューから次が行えます
  - 結晶を複製
  - 色の変更
  - 結晶削除

スマホでは、面ごとのカードで編集します。

- `h / k / i / l`
- 係数
- 色
- 面の表示切り替え
- 面文字の開閉
- 等価な面の追加
- 面の削除

### 各行でできること

- 面の表示切替
- ミラー指数の編集
- 面係数の編集
- 面色の設定
- 等価面の作成
- 面の削除
- 文字加工設定の開閉

### 文字加工

各面で次を設定できます。

- 文字列
- フォント
- 文字サイズ
- 深さ
- 横位置
- 縦位置
- 回転角

![面一覧で面文字設定を開き、刻印文字やフォント、文字サイズ、深さを編集している状態。](./images/user-manual/face-text-settings.png)

## 9. プレビュー

右上のプレビューで現在の構成を確認できます。

### 表示モード

右側カードの上部で表示モードを選べます。

たとえば次のようなモードがあります。

- 等価面
- 単色
- グレー
- 半透明
- カスタム(β版)

### 表示トグル

右側カードでは、次の表示を切り替えられます。

- 面指数
- 稜線
- 交線
- 軸（内）
- 軸（外）
- 軸ラベル
- 双晶軸 / 双晶面ガイド
- 分割面
- ラベル
- 結晶ごとの表示

面文字が入っている面では、文字と面の境界が補助線として重なって表示されます。これは `交線` トグルとは独立して表示され、面文字の位置や大きさの確認に使えます。

### スマホでのプレビュー操作

スマホでは、プレビューの下端に小さい操作帯が出ます。

- `視点リセット`
- `回転の慣性`
- 軸方向ボタン

タッチ操作は次の前提です。

- 1本指で回転
- 2本指で拡大縮小

操作説明は、スマホではプレビューの外側に短い文として表示します。

### プレビュー詳細設定

面一覧カードの下部にある `プレビュー詳細設定` で、次を細かく調整できます。

- 線色
- 線幅
- 透明度
- ラベル色
- フォント
- カスタム表示設定

## 10. 保存と読み込み

### 保存

ヘッダーの保存メニューから次を選べます。

- STL
- SVG
- PNG
- JPEG
- JSON

スマホでは、主な保存と読み込みは `出力` タブにまとまっています。

- `プロジェクトデータ`
  - `JSON保存`
  - `JSON読込（全て / 結晶データ / プレビュー設定）`
- `3Dモデル`
  - `STL`
- `プレビュー画像`
  - `SVG / PNG / JPEG`

![スマホの出力タブ。JSON、STL、SVG、PNG、JPEG の保存ボタンがまとまっています。](./images/user-manual/mobile-output.png)

`STL分割（β版）` が有効な時に STL を保存すると、指定した面で分割した STL を保存します。

### JSON の読み込み

JSON 読み込みでは、用途に応じて次を選べます。

- 全て
- 結晶データ
- プレビュー設定

## 11. よくある使い方

### 単結晶を作る

1. プリセットを選ぶ
2. 結晶パラメーターを調整する
3. 面一覧で必要な面だけ残す
4. プレビューで確認して STL 保存

### 双晶を作る

1. 基準結晶を作る
2. `+` で派生結晶を追加する
3. 双晶タイプを選ぶ
4. 双晶面または双晶軸を入力する
5. 必要なら接触面参照を設定する
6. プレビューで見え方を確認する
7. STL / SVG / PNG / JPEG を保存する

### 面に文字を入れる

1. 面一覧で対象面を開く
2. `文字` を開く
3. 刻印文字 / フォント / 文字サイズ / 深さを設定する
4. プレビューで位置とサイズを確認する
5. 面上に重なる輪郭線で、実際の文字境界と面内の配置を確認する

## 12. 補足

- 複雑な双晶構成では、警告が出ることがあります
- プリセットは初期値の出発点として使い、その後に手で調整できます
- 日本語 UI を基準にしていますが、英語 UI へ切り替えることもできます
